# Informação!
Nesta aula, vamos aprender e executar um protocolo completo para preparar, simular e analisar os resultados de uma simulação de dinâmica molecular de um complexo proteico anticorpo-antígeno, utilizando o programa AMBER. Como exemplo, usaremos o complexo formado pela proteína de morte celular programada 1 (PD-1) — que desempenha um papel crucial no escape das células tumorais do sistema imune — e o anticorpo Pembrolizumab, que inibe essa função da PD-1.

Ao final desta prática, você será capaz de:
+ Preparar uma estrutura para uma simulação de dinâmica molecula        
+ Explicar cada etapa de uma simulação        
+ Pós-processar uma simulação      
+ Analisar e plotar seus resultados com as ferramentas como CPPTRAJ e Python usando o Google Colab
